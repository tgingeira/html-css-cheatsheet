CSS Selectors

Basic Selectors

Selector Example Description
Element p {
 color: blue;
}		 Selects all <p> elements
Id #id {
 color: blue;
}		 Selects all elements with the id ID
Class .class {
 color: blue;
}		 Selects all elements with the class code
Universal Selector * {
 color: blue;
}		 Selects all html elements
Grouping Selector h1, h2, p {
 color: blue;
}		 Selects all <h1>, <h2> and <p> elements
Attribute [attribute] {
 color: blue;
}		 Selects all elements with the attribute attribute

Combinator Selectors

Selector Example Description
Descendant Selector div p {
 color: blue;
}		 Selects all <p> elements inside <div> elements
Child Selector div > p {
 color: blue;
}		 Selects all <p> elements where the parent is a <div> element
Adjacent Sibling Selector div + p {
 color: blue;
}		 Selects all <p> elements that are placed immediately after <div> elements
General Sibling Selector div ~ p {
 color: blue;
}		 Selects all <p> elements that are placed after <div> elements
